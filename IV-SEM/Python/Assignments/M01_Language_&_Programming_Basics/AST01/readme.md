# Ticket Pricing

## Problem Statement
Write a program for a cinema that asks for age and then calculates the price:
'''
Age < 5: Free
Age 5–17: $10
Age 18–64: $20
Age 65+: $15
'''

### Example
Input:16

Output:
10
